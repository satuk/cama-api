# API

```javascript       
GET  /api/events
```

Retrieves a list of all Events. 

```javascript
GET /api/events/{id}
```
Retrieves one specific Event with all the information.

```javascript       
GET  /api/companies/
```

Retrieves a list of all Companies.

```javascript
GET /api/companies/{id}
```
Retrieves one specific Company with all the information.
